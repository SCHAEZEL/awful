AWF's Utility Library

A collection of mostly matlab utilities. See MatlabAllHelp for the autogenerated matlab help files.

Symbolic toolbox helpers

au_coeff Coefficients of symbolic polynomial
au_ccode Convert symbolic expression to C with common subexpression elimination
au_autodiff_generate Symbolic differentiation and code generation

Faster alternatives to matlab builtins

au_sparse Create spare matrices more efficiently
au_whist Weighted histogram
au_bsx Cleaner interface to bsxfun

Printing and testing

au_prmat Compact matrix printing
au_test* Utilities for writing unit tests
au_assert* Easier assertions

Optimization/Numerics

au_levmarq Simple Levenberg-Marquardt
au_ransac Simple RANSAC
au_rosenbrock Rosenbrock test function
au_logsumexp Numerically more stable version of log(sum(exp(V)))

Vision/Graphics/Machine Learning

au_rodrigues Exponential map to rotation matrix via Rodrigues's formula
au_cross_matrix 3x3 matrix effecting cross product

MEX helper

The library also contains au_mex.h, a collection of C++ classes which greatly simplify mex file writing.

Note

The au_ prefix is because it's important in matlab's flat namespace that clashes of function names are avoided.
